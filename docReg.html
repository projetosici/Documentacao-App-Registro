1. Introdução

O software de registro de pacientes oncológicos pediátricos é uma ferramenta desenvolvida para auxiliar profissionais de saúde no atendimento e acompanhamento de pacientes. Ele permite o registro de informações relevantes sobre os pacientes, como dados demográficos, histórico médico, data do encaminhamento de tratamento, data de início do tratamento, estadiamento, data de óbito, entre outros. Além disso, ele permite uma visão mais atraente e menos poluída aos registradores e demais profissionais que venham a utilizá-lo.

1. Visão Geral do Software

O nosso inovador software de registro de pacientes é uma aplicação web moderna, desenvolvida com a poderosa tecnologia Flutter. Esta plataforma se integra perfeitamente com a API do RedCap, garantindo que os dados dos pacientes sejam acessados e armazenados com a máxima segurança em um banco de dados robusto. A interface de usuário, também criada em Flutter, foi projetada com o objetivo de proporcionar uma experiência intuitiva e eficiente para os profissionais de saúde. Com ela, médicos e enfermeiros e registradores podem acessar e atualizar as informações dos pacientes de maneira rápida e fácil, melhorando significativamente o fluxo de trabalho e a qualidade do atendimento.

3. Funcionalidades Principais

  • Registro de pacientes com informações detalhadas: Permite o registro de informações sobre os pacientes, como nome, idade, sexo, endereço, telefone, e-mail, histórico médico, tratamentos realizados e resultados de exames.
  • Envio de lembretes: Envia lembretes automáticos para os profissionais sobre seus pacientes. Para atualizar e acompanhar os casos registrados.
  • Acesso seguro: Garante o acesso seguro às informações dos pacientes por meio de autenticação e autorização.
  • Integração com RedCap
  • Segurança de dados sensíveis

7.6. RedCap - Banco de Dados:

RedCap (Research Electronic Data Capture) é utilizado como nosso banco de dados principal. RedCap é uma plataforma segura e robusta, amplamente utilizada para gestão de dados de pesquisa e informações de pacientes. Ele oferece funcionalidades avançadas para armazenamento, gestão e análise de dados.

7.7. Uso da API do RedCap:

A API do RedCap é utilizada para integrar diretamente o aplicativo ao banco de dados. Através desta API, é possível acessar, salvar e atualizar informações dos pacientes de forma segura. A API permite que o aplicativo comunique-se diretamente com o RedCap, garantindo que todas as operações sejam realizadas de maneira eficiente e com integridade dos dados.

12. LGPD

Lei Geral de Proteção de Dados (LGPD): Este software está em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), que regula o tratamento de dados pessoais no Brasil. A LGPD estabelece diretrizes para a coleta, armazenamento, processamento e compartilhamento de dados pessoais, garantindo a privacidade, a segurança e a transparência no tratamento dessas informações. Este software foi desenvolvido com o objetivo de proteger os dados pessoais dos usuários, garantindo o consentimento informado e transparente para o uso desses dados, bem como a possibilidade de acesso, correção e exclusão dessas informações conforme solicitado pelo usuário. Além disso, o software registra e rastreia todas as atividades relacionadas aos dados pessoais, garantindo a prestação de contas e a responsabilidade do controlador dos dados.
